# 并行任务自动排程器 Android APK

这是一个 WebView Android 项目，`app/src/main/assets/index.html` 是当前最新版任务排程器页面。

## GitHub 在线打包 APK

1. 把本项目所有文件上传到 GitHub 仓库根目录。
2. 进入仓库的 Actions 页面。
3. 运行 `Build Android APK`。
4. 构建完成后，在 Artifacts 里下载 `ParallelTaskScheduler-debug-apk`。
5. 解压得到 `app-debug.apk`，安装到安卓手机。

这是 debug APK，手机安装时可能需要允许“安装未知来源应用”。
